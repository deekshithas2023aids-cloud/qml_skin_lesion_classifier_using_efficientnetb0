Skin Lesion Classification using Quantum Machine Learning and EfficientNet
 Project Overview
This project presents an intelligent skin lesion classification system developed using Quantum Machine Learning (QML) integrated with the EfficientNet deep learning architecture. The system is designed to classify skin lesion images accurately and assist in the early detection of skin-related diseases.
The project leverages the power of deep learning and quantum computing concepts to improve classification performance and computational efficiency in medical image analysis.

 Technologies Used
- Python
- Google Colab
- TensorFlow
- Keras
- EfficientNet
- Quantum Machine Learning (QML)
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

 Features

- Deep learning-based feature extraction
- Quantum-enhanced learning approach
- High classification accuracy
- Medical image preprocessing support
- User-friendly implementation in Google Colab
